1. <select>:Defines a drop-down list
  2. <optgroup>:Defines a group of related options in a drop-down list
  3. <option>:Defines an option in a drop-down list
  4. <label>:Defines a label for an element
  5. <fieldset>:Groups related elements in a form
  6. <legend>:Defines a caption for a fieldset input element
  7. <img>:Defines an image
  8. <map>:Defines a client-side image map
  9. <area>:Defines an area inside an image map
  10. <canvas>:Used to draw graphics, on the fly, via scripting (usually JavaScript)
  11. <figcaption>:Defines a caption for a element
  12. <figure>:Specifies self-contained content
  13. <picture>:Defines a container for multiple image resources
  14. <svg>:Defines a container for SVG graphics
  15. <audio>:Defines sound content
